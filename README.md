# Simply Ummiby Workshop

**Version 0.8.0**

Version 0.8.0 refreshes Inventory as a workshop stockroom dashboard while preserving the existing catalog, suppliers, prepared-component, product-link, archive/delete, and transaction workflows.

## Version 0.8.0 highlights

- Adds a warm Inventory shelving banner that matches the new Orders design language.
- Adds live summary cards for total catalog items, low stock, prepared components, and the Restock Center.
- Adds Quick Actions for New Inventory Item, Receive Stock, Adjust Inventory, Restock Center, and printing the inventory list.
- Adds a compact Low Stock panel beside the full inventory table.
- Keeps the existing material-type and operational tabs as table filters.
- Adds a dedicated Receive Stock flow that increases an existing item and records a transaction.
- Preserves suppliers, linked products, purchase links, material-type tabs, prepared dowels, safe archive/delete, and inventory history.
