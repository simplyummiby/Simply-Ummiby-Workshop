# Changelog

## 0.5.2 — 2026-07-14

### Changed

- Widened application dialogs on desktop
- Added a scrollable modal content region
- Added a sticky modal header and sticky action footer
- Save and Cancel actions remain visible while long forms are scrolled
- Improved responsive spacing and mobile dialog behavior
- Expanded inventory editor form space and image preview area

## 0.5.1 — 2026-07-14

### Changed

- Removed broad quantity summary cards
- Raw Materials, Packaging, and Print Supplies now use table-style views
- Fabrication Kits and Finished Inventory retain cards

### Added

- Search
- Craft, type, and stock filters
- Alphabetical, quantity, and urgency sorting
- Product, color, and stock grouping
- Optional item photos with local compression
- Craft and material-type metadata

## 0.5.0 — 2026-07-14

- Added the full Inventory Catalog
