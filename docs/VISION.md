# Vision

## Simply Ummiby Workshop

Simply Ummiby Workshop is a personal workshop management system built around the actual production workflow of the Simply Ummiby Etsy shop.

It does not attempt to replace Etsy, Shippo, Cricut Design Space, or supplier websites. Instead, it organizes the work between those tools:

- preparing materials,
- managing fabrication kits,
- making products,
- packing orders,
- printing resources,
- watching inventory,
- and deciding what needs to be purchased, made, or printed next.

## Product principles

1. **Workshop-first, not marketplace-first.**
2. **Local and private before cloud-connected.**
3. **Fast enough to use while actively making.**
4. **Every physical item can move independently.**
5. **One customer order can contain several items and several packages.**
6. **Inventory deductions should reflect real workshop actions.**
7. **The system should reduce thinking, searching, and forgotten supplies.**
8. **The interface should feel warm, handmade, and welcoming.**

## Long-term direction

The Workshop may later support:

- Google Drive backups,
- readable Google Sheets exports,
- Etsy order import,
- cross-device cloud sync,
- deeper recipe management,
- reporting,
- and cost tracking.

These additions should never make daily workshop use feel complicated.
